# Expense Tracker Python

A Python-based expense tracking application used to manage and monitor daily expenses efficiently.

## Features
- Add expenses
- Update expense records
- Delete expenses
- View expense history
- Expense categorization
- Data analysis using SQL queries

## Tech Stack
- Python
- SQLite / MySQL
- SQL
- Git

## Concepts Used
- CRUD Operations
- Database Management
- SQL Queries
- Input Validation
- Error Handling

Project under development.
